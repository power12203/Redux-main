# 포트폴리오 예시 - Redux를 활용한 상품 관리

이 프로젝트는 Redux를 사용하여 간단한 상품 관리 기능을 구현한 예시입니다. Redux를 이용하여 상태를 관리하고 React 컴포넌트에서 이를 활용하는 방법을 보여줍니다.

## 기술 스택

- React
- Redux
- styled-components

## 설치 및 실행 방법

```bash
# 프로젝트 클론
git clone https://github.com/your/repository.git

# 프로젝트 폴더로 이동
cd project-folder

# 의존성 설치
npm install

# 애플리케이션 실행
npm start

```
